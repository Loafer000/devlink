Setup CI/CD Pipeline for a Node.js Application

DevOps Posted 2 days ago

Project Description

We are seeking an experienced DevOps engineer to design and implement a robust Continuous Integration/Continuous Deployment (CI/CD) pipeline for our primary Node.js application. The application is hosted on Amazon Web Services (AWS), and we utilize GitHub for our version control.

The main objective is to automate our software delivery process, allowing for faster and more reliable deployments. The ideal candidate will have a strong background in AWS, GitHub Actions, and containerization technologies like Docker.

Key Responsibilities:

  • Configure GitHub Actions workflows for building, testing, and packaging the application.
  • Automate deployment to AWS services (e.g., EC2, ECS, or Lambda).
  • Implement automated testing suites (unit, integration) within the pipeline.
  • Set up monitoring and alerting for the pipeline and deployed application.
  • Provide documentation and guidance to our development team.
